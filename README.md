# DAWN RU

Неофициальная русская локализация **DAWN: The RPG** Джоэла Врёгденхила, инструменты сборки книги, офлайн-компаньон и учебные материалы.

Текущая версия: **0.9.0**.

## Структура

- `source/original/` — исходный англоязычный PDF.
- `source/translation/` — канонический русский текст, глоссарий и редакторский стиль.
- `source/editorial/` — история редакторских сверок.
- `tools/layout/` — сборщик и ресурсы русской верстки.
- `apps/companion/` — автономный компаньон 2: сборка героя, игровой стол, XD6, часы и справочник.
- `media/tutorials/` — 12 адаптивных объясняющих анимаций с единым проигрывателем.
- `release/` — только канонические релизные файлы.
- `build/` — воспроизводимые промежуточные результаты.
- `archive/` — старые сборки, пакеты и экспериментальные материалы.

## Сборка книги

Требуется Python 3.12 или новее.

```powershell
python -m pip install -r requirements.txt
python tools/layout/build_pilot_pdf.py final
```

Результат: `release/DAWN-RU-v0.9.0.pdf`.

## Обновление данных компаньона

```powershell
python apps/companion/build_data.py
cd apps/companion
npm test
```

Компаньон читает канонический текст из `source/translation/`. Для запуска без установки откройте
`apps/companion/index.html`. Он поддерживает старые сохранения, несколько героев, печать и переносимый JSON.

## Учебные анимации

Откройте `media/tutorials/00_оглавление.html`. Каждый урок можно ставить на паузу,
перезапускать и перелистывать; холст корректно масштабируется от телефона до 1920×1080.

## Редакторская политика
