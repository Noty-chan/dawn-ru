# DAWN RU · Компаньон 2

Автономное приложение для сборки героя и игры за столом. Для обычного использования достаточно
открыть `index.html` двойным кликом. При запуске через локальный сервер приложение также
устанавливается как PWA и кэширует себя для офлайн-работы.

## Главное

- версия сохранений 2 со стабильными id и миграцией старого ключа `dawn-heroes`;
- корректное округление Скорости и Фокуса вверх;
- рост Ступеней, обмен двух Уровней Техник на высший Атрибут, до трёх Мировоззрений;
- особые бюджеты Даров `Past Your Prime`, `Amazing Potential`, `Supernatural Deafness`,
  `Dark Urge`, `Uncontrollable Power`, `Gearhead` и `Tainted Body`;
- конструктор Способности из Глаголов, Существительных и Условий канонического глоссария;
- игровой стол с раздельными Ранами от себя и извне, эффектами, заметками и фокус-режимом;
- XD6-бросок со взрывающимися шестёрками и Ва-банк, история бросков и часы сцены;
- единый поиск по памяткам, действиям, эффектам, Дарам и всем 107 Техникам;
- адаптивный интерфейс без горизонтального переполнения на ширине 320 px;
- безопасный импорт ограниченного JSON, экспорт, печать и несколько героев.

## Обновление данных

Из каталога `apps/companion`:

```powershell
python build_data.py
npm test
```

Генератор читает канонические Markdown-файлы из `source/translation` и создаёт `data.js`.
Этот файл вручную не редактируется.

## Состав

- `index.html` — семантическая оболочка;
- `app.css` — адаптивная тема и печатный лист;
- `app.js` — состояние, расчёты, стол и инструменты;
- `data.js` — автогенерируемые правила;
- `build_data.py` — извлечение данных из перевода;
- `manifest.webmanifest`, `sw.js` — устанавливаемый офлайн-режим;
- `tests/qa.mjs` — проверка полноты данных, id и ключевых механических инвариантов.
